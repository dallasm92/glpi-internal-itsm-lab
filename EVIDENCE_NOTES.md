# Evidence Notes

## Screenshot Scope

The source screenshot folder for this lab included both GLPI images and unrelated ServiceNow captures. Only the GLPI-specific screenshots were copied into this repository.

## What The Evidence Covers

The curated screenshots show:

- the working GLPI dashboard
- the `MAIN-PC` asset record
- notes attached to the `MAIN-PC` asset
- the first incident ticket lifecycle
- the first ITIL category set

## What The Evidence Does Not Show

The current screenshot set does not directly show:

- container creation on `pi-core`
- MariaDB configuration
- Pi-hole DNS record creation
- backup script creation or cron configuration

Those parts of the lab are documented from the deployment session notes and live system state rather than browser screenshots.

## Safety Notes

- This repo uses only internal lab evidence and avoids publishing sensitive credentials.
- The service remains documented as internal-only at `http://glpi.lan:8088`.
